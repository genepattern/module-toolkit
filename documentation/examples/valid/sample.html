CorrelationMatrix Module

This module calculates correlation matrices from gene expression data.

Parameters

  • input - Input GCT file containing gene expression data
  • method - Correlation method (pearson, spearman, kendall)
  • dimension - Calculate correlations across rows or columns
  • output - Output file name for the correlation matrix

Usage

The CorrelationMatrix module can be used to analyze relationships between genes or samples.

Examples

Use with default Pearson correlation:

CorrelationMatrix input=data.gct method=pearson dimension=column output=correlation.gct
